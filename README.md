* Android Studio or VS Code with Flutter extension
* Emulator or physical device

### Steps:

1. Clone the repo:

   ```bash
   git clone 'https://github.com/L-Semakale/linear_regression_model'
   cd diabetes-predictor-app
   ```

2. Get dependencies:

   ```bash
   flutter pub get
   ```

3. Launch the app:

   ```bash
   flutter run
   ```
